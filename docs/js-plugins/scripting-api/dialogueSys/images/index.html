images

class in dialogueSys.js / Implemented in dialogueSys

Description

The class that holds images.
Contains a 'class for preloaded images'.

Static Properties

isActive (Read Only) Defines whether the images can be used.
directory (Read Only) The source directory of every image.
images (Read Only) The list of every image in this system.

Static Methods

Set Sets the images by the given profiles.
Find Finds an image by its name.
changeDir Changes the source directory of every image.
