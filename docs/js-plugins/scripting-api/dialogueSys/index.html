dialogueSys

class in dialogueSys.js

Description

A class that creates a div based RPG style dialogue system.

Static Properties

PROFILE_DEFAULT Used for setting the default data profile.
DIR_SAME Used for making the audios/images source directory the same to the window's current location.
PORTRAIT_LEFT Used for making the dialogue portrait position left.
PORTRAIT_RIGHT Used for making the dialogue portrait position right.

Properties

isLoaded (Read Only) Defines whether the system is loaded.
isActive (Read Only) Defines whether the system is active.
isInUse (Read Only) Defines whether this has an active process or in use.
dialogueBox (Read Only) The dialogue box element.
dialogueLine (Read Only) The dialogue text line element.
dialoguePortrait (Read Only) The dialogue portrait image element.
profiles The data profiles of this system.
charCount (Read Only) Tells how many chars are in the dialogue line.
portraitPosition The current portrait position.
timeScale The scale which time passes.
currentEvent (Read Only) The current event that is happening.

Classes

style The style of every char that will be appended. Similar to 'CSSStyleDeclaration' class, but does not work the same.
animations The animations of every element.
audios The class that holds audios.
images The class that holds images.

Methods

on Adds/Removes a callback for a specified event.
once Adds/Removes a callback that will be called once for a specified event.
finish Finishes the current process, but does not stop it.

Asynchronous Methods

delay Creates a delay that has precision but is affected by a time scale.
SetActive Enables/Disables the dialogue system.
type Types/uses the dialogue system.
clearText Clears the dialogue text line.
clear Clears the dialogue text line and the dialogue portrait image.
