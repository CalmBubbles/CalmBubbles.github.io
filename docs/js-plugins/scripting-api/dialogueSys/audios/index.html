audios

class in dialogueSys.js / Implemented in dialogueSys

Description

The class that holds audios.
Contains a 'class for preloaded audios'.

Static Properties

isActive (Read Only) Defines whether the audio update system is active.
volume The max volume of every audio. Default is '-1' for no limit.
directory (Read Only) The source directory of every audio.
audios (Read Only) The list of every audio in this system.

Static Methods

Set Sets the audios by the given profiles.
Find Finds an audio by its name.
changeDir Changes the source directory of every audio.
